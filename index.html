Dropdowns

Searchable

Add 'selectpicker' class to the select element and add data-live-search="true"
<select class="form-control selectpicker" data-live-search="true"></select>

Non-Searchable

Remove 'selectpicker' class to the select element and add data-live-search="true"
<select class="form-control"></select>

Searchable (multiple)

Add 'selectpicker' class to the select element, data-live-search="true" and mutiple
<select class="form-control selectpicker" data-live-search="true" multiple></select>

Non-Searchable (multiple)

Remove 'selectpicker' class to the select element and data-live-search="true"
<select class="form-control" mutiple></select>
